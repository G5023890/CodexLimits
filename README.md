# CodexLimits

Небольшое macOS menu bar приложение для автоматического отображения лимитов Codex.

## Что делает

- Показывает native menu bar индикатор с круговым usage ring и компактным процентом.
- По нажатию открывает popover с:
  - большим круговым gauge для недельного лимита;
  - остатком лимита `5h`;
  - stats для токенов / запросов, если их отдаёт локальный `Codex.app` app-server;
  - plan name и reset timer.
- Читает данные автоматически из локального `Codex.app` через внутренний `app-server`.

## Логика состояний

- `40...100%` - normal / green
- `15...39%` - warning / yellow
- `0...14%` - critical / red

- Menu bar индикатор использует недельный лимит как основной.
- Обновление выполняется автоматически каждые `60` секунд и при открытии popover.

## Сборка и установка

```bash
./scripts/build_and_install_app.sh
```

После установки приложение появится в `/Applications/CodexLimits.app`.
